# STM32 CAN introduction
"Standards and Communication Systems" course - introduction to CAN protocol using Nucleo boards

Used boards:
- Nucleo-F103RB
- Nucleo H7A3ZI-Q

Additional hardware used:
- 2x 3.3V CAN transceiver boards based on SN65HVD230 IC
- Analog Discovery 2
